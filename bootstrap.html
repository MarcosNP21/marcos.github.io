Framework: Bootstrap

Bootstrap es el framework CSS más popular. Facilita la creación de sitios con un sistema listo para usar. Es fácil de aprender y su principal inconveniente es que muchos sitios se parecen si no se personaliza y puede incluir más código del necesario.
